## ✨ Overview

ComicPreneur is a modern, high-performance web platform built to manage event registrations with a seamless user experience and a powerful admin system.  

From smooth animations to real-time database integration, this project blends **design, performance, and scalability** — making it production-ready.

---

## 🎯 Key Highlights

- ⚡ Lightning-fast React (Vite) frontend  
- 🎨 Smooth UI/UX with GSAP animations  
- ☁️ Firebase-powered backend (Firestore + Storage)  
- 📊 Admin dashboard with Excel export  
- 📱 Fully responsive across all devices  
- 🔥 Production deployed on Firebase Hosting  

---

## 🧩 Features Breakdown

### 👤 User Experience

- Smart registration form  
- File/image upload support  
- Real-time submission handling  
- Clean and interactive UI  

### 🔐 Admin Panel

- Secure admin login  
- View all registrations instantly  
- Export data to Excel (XLSX)  
- Scalable structure for future upgrades
